NAME
----
cdist-type__apt_update_index - update apt's package index


DESCRIPTION
-----------
This cdist type runs apt-get update whenever any apt sources have changed.


REQUIRED PARAMETERS
-------------------
None.

OPTIONAL PARAMETERS
-------------------
None.
